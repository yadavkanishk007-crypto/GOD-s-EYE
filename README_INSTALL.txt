Thank you for installing the GOD's EYE Urban Intelligence System.

Quick Start:
1. Launch "GOD's EYE" from your Start Menu or Desktop.
2. The application will start immediately. If it's your first time, the YOLOv8 AI model will be automatically configured.
3. Click "Add Camera" to connect an RTSP stream, USB webcam, or local video file.
4. Go to the "Track" tab to upload a photo and use the Person Re-Identification (Re-ID) system.

Data Privacy:
- All video streams are processed completely offline on your local CPU.
- No biometric or surveillance data ever leaves your machine.
- Event clips are stored locally in the `data/events` directory within the installation folder and are automatically pruned after 30 days.

Troubleshooting:
- If you encounter permission errors, ensure you run the application as Administrator.
- To reset configuration, modify the `config.yaml` file located in the installation folder.

For technical support, contact the Indian Urban Intelligence deployment team.
